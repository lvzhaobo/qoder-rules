Form API 参数说明

参数 说明 类型 默认值
inline 内联表单 Boolean -
size 单个 Item 的 size 自定义 Enum: 'large', 'medium', 'small' 'medium'
labelAlign 标签的位置 Enum: 'top', 'left', 'inset' 'left'
field field 实例 any -
onSubmit 提交表单时触发 Function preventDefault
device 预设屏幕宽度 Enum: 'phone', 'tablet', 'desktop' 'desktop'

Form.Item API 参数说明

参数 说明 类型 默认值
label label 标签的文本 ReactNode -
required 不能为空 Boolean -
help 自定义提示信息 ReactNode -
validateState 校验状态 Enum: 'error', 'success', 'loading', 'warning' -
hasFeedback 是否展示校验状态图标 Boolean false
format 四种常用的 pattern Enum: 'number', 'email', 'url', 'tel' -

说明

Form 组件来自于 https://gitee.com/tongyilingma/ui-components-wiki/blob/master/basic/form.md

表单组件用于数据收集、校验和提交功能。支持多种布局方式，包括水平布局、垂直布局和行内布局。

表单校验可以使用内置规则（如required、email等）或自定义校验函数实现。
