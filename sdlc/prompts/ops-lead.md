# 运维组长提示词 (Operations Team Lead)

## 角色定位
**运维组长** - 负责基础设施审计、安全合规和成本优化

---

## 场景: 基础设施审计 & 成本优化

### 目标
审计 K8s 配置的安全与成本

### 提示词

```
请严格审计这份 Kubernetes Deployment YAML：
1. **安全隐患**：容器是否以 root 运行？是否限制了 Capabilities？
2. **资源设置**：Requests/Limits 设置是否合理？（防止 OOM 或 资源浪费）
3. **高可用**：是否配置了 LivenessProbe 和 PodDisruptionBudget？
4. **标签规范**：Labels 是否符合 [公司标准，如 app, team, env]？

请输出审计报告，并对高风险项标红。
```

### 预计输出
- 安全与稳定性风险清单
- 资源配置优化建议
- 标签规范合规性检查

### 效果
✅ 消除基础设施安全隐患，提升资源利用率，方便成本分摊统计。

---

## 使用建议

💡 **适用场景：**
- K8s 配置审计
- 容器安全检查
- 资源成本优化
- 高可用性评估

**注意事项：**
- 容器不要以 root 用户运行，遵循最小权限原则
- 资源限制要合理设置，避免 OOM 或资源浪费
- 健康检查探针要配置准确，避免误杀正常容器
- 标签规范要统一，方便资源管理和成本分摊
- 定期审计配置，确保符合安全基线
